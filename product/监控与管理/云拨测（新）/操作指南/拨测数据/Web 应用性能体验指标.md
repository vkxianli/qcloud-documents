## 页面性能指标

| 指标名称              | 定义和算法                                                   | 作用                                                         | 单位    | 评估标准-较好 | 评估标准-一般 | 评估标准-较差 |
| --------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------- | ------------- | ------------- | ------------- |
| 整体性能              | 从页面开始浏览到最后一包数据接收完成之间的时间间隔           | -       | <nobr>秒（s）</nobr> | <=10          | <=20          | >20           |
| 100K 耗时             | 平均加载100KB内容所需耗时，100K 耗时 = 整体性能 / 总下载字节数 × 100% | 更加科学公正的展现不同页面加载耗时                           | 秒（s） | <=0.5         | <=1           | >1            |
| 文档完成用时          | 从页面开始浏览到基础文档解析完成之间的时间间隔               | 从文档解析角度衡量页面完成的性能                             | 秒（s） | <=10          | <=20          | >20           |
| 渲染用时              | 页面除基础文档之外，其它元素加载完成所用的耗时。渲染用时 = 整体性能 - 基础文档下载用时 | 衡量页面代码质量                                             | 秒（s） | <=10          | <=20          | >20           |
| 整体速度              | 页面的平均加载速度     。整体速度 = 总下载字节数 / 整体性能  | -            | KB/s    | >150          | >80           | <=80          |
| 渲染速度              | 页面基础文档下载完毕之后，加载和渲染页面其它元素的速度。<li>渲染量 = 总下载字节数－基础文档下载字节数<li>     渲染用时 = 整体性能－基础文档下载用时  <li>   渲染速度 = 渲染量 / 渲染用时 | -        | KB/s    | >180          | >100          | <=100         |
| 基础文档速度          | 基础文档速度  = 基础文档下载字节数/基础文档下载用时          | -          | KB/s    | >220          | >100          | <=100         |
| 投影DNS用时           | 所有元素 DNS 用时之和（重合时间算一次，元素之间的间隔时间不算在内） | -        | 秒（s） | <=0.8         | <=1.8         | >1.8          |
| 投影TCP用时           | 所有元素 TCP 用时之和（重合时间算一次，元素之间的间隔时间不算在内） | -      | 秒（s） | <=1.5         | <=2.5         | >2.5          |
| 投影响应用时          | 所有元素响应用时之和（重合时间算一次，元素之间的间隔时间不算在内） | -    | 秒（s） | <=3.5   | <=8           | >8     |
| 投影下载用时          | 所有元素下载用时之和（重合时间算一次，元素之间的间隔时间不算在内） | -      | 秒（s） | <=6           | <=12          | >12           |
| 投影 BLOCK用时        | 所有元素响应 BLOCK 用时之和（重合时间算一次，元素之间的间隔时间不算在内） | -       | 秒（s） | <=4           | <=10          | >10           |
| 投影SSL握手用时       | 所有元素 SSL 握手用时之和（重合时间算一次，元素之间的间隔时间不算在内） | -      | 秒（s） | <=1.5         | <=2.5         | >2.5          |
| 基础文档 DNS 用时     | 下载基础文档元素的 DNS 查询用时                              | -       | 秒（s） | <=0.1         | <=0.2         | >0.2          |
| 基础文档 TCP 用时     | 下载基础文档元素的 TCP 查询用时                              | -      | 秒（s） | <=0.12        | <=0.25        | >0.25         |
| 基础文档响应用时      | 下载基础文档元素的等待响应用时                               | -      | 秒（s） | <=0.3         | <=0.8         | >0.8          |
| 基础文档下载用时      | 下载基础文档元素接收数据用时，即从收到服务器响应第一包数据到基础文档最后元素最后一包下载完成时刻。 | -           | 秒（s） | <=1           | <=2.5         | >2.5          |
| 基础文档 SSL 握手用时 | 下载基础文档元素的 SSL 握手用时                              | -           | 秒（s） | <=0.5         | <=1           | >1            |
| 平均 DNS 用时         | 所有元素平均 DNS 花费时间（Cache 元素除外）                  | -        | 秒（s） | <=0.15        | <=0.3         | >0.3          |
| 平均 TCP 用时         | 所有元素平均 TCP 花费时间（Cache 元素除外）                  | -      | 秒（s） | <=0.2         | <=0.5         | >0.5          |
| 平均响应用时          | 所有元素平均响应花费时间（Cache 元素除外）                   | -        | 秒（s） | <=0.3         | <=0.8         | >0.8          |
| 平均下载用时          | 所有元素平均下载花费时间（Cache 元素除外）                   | -       | 秒（s） | <=1           | <=2.5         | >2.5          |
| 平均 SSL 握手用时     | 所有元素平均 SSL 握手花费时间（Cache 元素除外）              | -        | 秒（s） | <=0.8         | <=1.6         | >1.6          |
| 首屏用时              | 从输入 URL 开始到页面已渲染区域高度大于等于指定高度的时间差，平台默认是600像素高。当页面不足600像素高度时，取开始浏览到 IE 内核抛出 Document  Completed 事件之间的时间差 | 从用户体验角度衡量页面渲染效果                               | 秒（s） | <=3.5         | <=7           | >7            |
| 可用性                | 执行监测任务的客户端对目标访问的成功率。可用性 = 有效监测次数 / 总监测次数 × 100% | -         | %       | >96           | >92           | <=92          |
| 首屏完全渲染用时      | 页面指定范围内所有元素首次被渲染完毕的时间                   | 真实反映用户感受渲染效果的性能指标                           | 秒（s） | <=6           | <=12          | >12           |
| 慢速比                | 指定时间范围内，整体速度低于正常值（该阈值可依据业务需求进行设置）的次数占有效监测次数的百分比。散点图不支持此指标查看 | 反映监测任务下载速度情况          | %       | <=20          | ＜50          | ＞50          |
| 首次渲染时间          | 从页面开始浏览到页面出现第一个像素之间的时间间隔             | -         | 秒（s） | <=0.05        | <=0.15        | >0.15         |
| DOMREADY 时间         | 从基础文档下载完成到基础文档可以解析之间的时间间隔           | DOM  Ready 时间如果过长，用户会发现页面已经加载完毕，但是很多功能不可用，可能单击某个链接会跳到页面顶部。因此，DOM  Ready 时间越短越好，则交互功能才能尽早可用 | 秒（s） | <=0.2         | <=0.5         | >0.5          |
| ONLOAD 时间           | 从页面开始浏览到基础文档中的所有静态资源解析且渲染完成之间的时间间隔 | js 动态加载的资源不计入基础文档相关的资源内。IE 9以下版本取不到 onload 时间 | 秒（s）  | <=0.5         | <=1           | >1            |
| 总下载字节数          | 整个浏览过程中 IE 内核的总下载量                             | -       | KB      | <=900         | <=2000        | >2000         |
| 基础文档下载字节数    | 页面的基础文档元素的大小                                     | -       | KB      | <=150         | <=300         | >300          |
| DNS 解析次数          | 整个浏览过程中一共发生的 DNS 查询次数                        | -   | 次      | <=30          | <=50          | >50           |
| TCP 建连次数          | 整个浏览过程中一共发生的 TCP 建立连接次数                    | -  | 次      | <=50          | <=90          | >90           |
| RoundTrips            | 整个测试过程和远程服务器的交互次数                           | -        | 个      | <=80          | <=150         | >150          |
| 平均元素个数          | 所有监测数据中平均加载完成的元素个数                         | -                                                            | 个      | <=80          | <=150         | >150          |
| 总错误元素个数        | 发生元素错误的元素总个数                                     | -                                                            | 个      | 无            | 无            | 无            |
| 首屏元素个数          | 首屏内加载完成的元素个数                                     | -                                                            | 个      | <=15          | <=30          | >30           |
