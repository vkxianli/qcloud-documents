- **日结客户**：截止2021年06月01日，云点播日结客户的低频存储/归档存储/深度归档存储 以及数据取回能力不收取任何费用。
- **月结客户**：请联系您的商务确认收费策略，免费只面向日结客户。

| 存储类型   | 抵扣比例    |
| ------ | ------- |
| 标准存储   | 1：1     |
| 低频存储   | 1：0.5   |

| 深度归档存储 | 1：0.125 |
